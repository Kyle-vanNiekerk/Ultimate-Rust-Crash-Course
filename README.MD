# Ultimate Rust Crash Course

Welcome to the Ultimate Rust Crash Course!  
This project is designed for educational purposes, providing hands-on exercises to help you learn Rust from the ground up.

## Purpose

- Learn Rust fundamentals through practical examples
- Build confidence with real code
- Suitable for beginners and those refreshing their Rust skills

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable version recommended)
- Basic command line knowledge

## Project Structure

- `exercises/` — Contains individual exercise folders
  - Each exercise has its own `Cargo.toml` and `src/main.rs`
- `README.MD` — This file

## How to Use

1. Clone this repository
2. Navigate to an exercise folder:
   ```sh
   cd exercises/exercise_a_variables
   ```
3. Run the exercise:
   ```sh
   cargo run
   ```
4. Edit the code, experiment, and learn!
